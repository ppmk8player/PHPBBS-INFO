PHPBBS

PHPだけで動作するシンプルな掲示板

📦 GitHub リポジトリ

この掲示板は、PHPのみで動作し、データベースは不要です。ファイルへの書き込みで投稿を保存します。

GitHubで見る

✨ 特徴

  • データベース不要（ファイル保存）
  • シンプルな構成で軽量
  • 投稿・削除機能付き
  • そのままアップロードして動作
  • HTML/CSSのカスタマイズも簡単

🚀 セットアップ方法

  1. リポジトリをクローンまたはダウンロード： 
    git clone https://github.com/ppmk8player/PHPBBS.git
  2. Webサーバーの公開ディレクトリに配置
  3. 書き込み権限を付与： 
    chmod -R 707 data
  4. ブラウザで `index.php` にアクセスして確認
